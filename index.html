Evolución del diseño y desarrollo web

Cronologia del diseño y desarrollo web

  • 1990: Nace el HTML, primer lenguaje de bloques básicos para construir sitios web.
  • 1991: Se creó el primer sitio web "World Wilde Web".
  • 1992: Diseño web basado en tablas.
  • 1994: Se estableció el W3C, World Wilde Web Consortium.
  • 1995-1996: Con la llegada de JavaScript (1995) y Flash (1996) se aporto interacción, animación y efectos especiales.
  • 1998: Se dio mayor dinamismo y simplicidad cuando de proporciono CSS y PHP.
  • 2003: Novedosas funcionalidades con CSS3 y WEB2 ademas de una era de la información basada en el usuario.
  • 2008: Diseño web para móviles o MOBILE SITIES.
  • 2010: Llega HTML5 que se volveria el nuevo estandar enriquesiendo los medios integrados y con mayor accesibilidad.
  • 2012: Surge el diseño responsivo para resolver los problemas con los distintos tamaños de pantalla en los nuevos dispositivos.
  • Actualidad: Uso de nuevas tecnologias como frameworks y librerias
